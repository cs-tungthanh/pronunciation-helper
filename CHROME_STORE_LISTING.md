## Detailed Description

**Pronunciation Helper - Your Personal English Pronunciation Coach**

Learning English pronunciation has never been easier! Pronunciation Helper is a powerful Chrome extension that instantly shows you how to pronounce any word or sentence you encounter on the web.

**🎯 Key Features:**

✓ **Instant Audio Playback** - Select any text and hear native American English pronunciation
✓ **IPA Notation** - See International Phonetic Alphabet transcription for every word
✓ **Smart Learning** - Automatically expands contractions for better understanding
✓ **Draggable Interface** - Move the popup anywhere to see content clearly
✓ **No Limits** - Works with single words, sentences, or entire paragraphs
✓ **100% Privacy** - No data collection, works offline for audio

**📖 Perfect For:**
- ESL/EFL students
- English language learners
- Teachers preparing materials
- Non-native speakers improving pronunciation
- Anyone encountering unfamiliar words

**🎨 How It Works:**

1. Highlight any text on any webpage
2. Popup appears instantly with phonetic notation
3. Click "Play" to hear pronunciation
4. Drag popup to move it out of the way
5. Click outside to dismiss

**🔊 Advanced Features:**

• Word-by-word IPA cards for detailed learning
• Smooth, natural-sounding text-to-speech
• Replay support during playback
• Copy IPA text for your notes
• Responsive, scrollable interface for long selections
• Smart detection prevents popup interference

**🛡️ Privacy & Permissions:**

This extension requires minimal permissions:
- **activeTab**: To read selected text
- **scripting**: To show the popup

We do NOT:
- Collect any user data
- Track your browsing
- Store your selections
- Share information with third parties

The extension works primarily offline using your browser's built-in voices. Only IPA data is fetched from a free, public dictionary API.

## Privacy Policy

**Privacy Policy for Pronunciation Helper**

Last updated: [Date]

**Data Collection:**
Pronunciation Helper does NOT collect, store, or transmit any personal data or browsing information.

**Permissions Used:**
- activeTab: Required to read text you select on webpages
- scripting: Required to display the pronunciation popup

**External Services:**
The extension fetches IPA phonetic data from the Free Dictionary API (https://dictionaryapi.dev/) when you select text. Only the selected words are sent to this service. No identifying information is transmitted.

**Audio:**
All text-to-speech audio is generated locally by your browser using the Web Speech API. No audio data is sent to external servers.

**Data Storage:**
No user data is stored locally or remotely.

**Third-Party Sharing:**
No user data is shared with any third parties.

**Children's Privacy:**
This extension does not knowingly collect information from children under 13.
